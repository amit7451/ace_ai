# ION AI Chatbot Platform

A production-grade, multi-tenant SaaS AI chatbot platform.

## Quick Start

1. Clone the repository.
2. Ensure you have Node.js 20+ and `pnpm` installed.
3. Run `docker-compose up -d` to start PostgreSQL and Redis.
4. Run `pnpm install`
5. Run `pnpm dev` to start all services locally.
